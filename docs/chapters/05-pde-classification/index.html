The general second-order linear PDE

Any linear second-order PDE in two variables can be written: $$A\,u_{xx} + B\,u_{xy} + C\,u_{yy} + D\,u_x + E\,u_y + F\,u = G$$

The discriminant

By analogy with conic sections (ellipse, parabola, hyperbola), the sign of $B^2 - 4AC$ classifies the PDE's behavior:

  • Elliptic: $B^2 - 4AC < 0$ — e.g., Laplace equation $\nabla^2 u = 0$
  • Parabolic: $B^2 - 4AC = 0$ — e.g., heat equation $u_t = \alpha u_{xx}$
  • Hyperbolic: $B^2 - 4AC > 0$ — e.g., wave equation $u_{tt} = c^2 u_{xx}$
